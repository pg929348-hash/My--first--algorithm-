# My First Algorithms 🏆

## greatest_second_greatest.py
First algorithm I built from scratch without using max() or sort().

**What it does:**
- Takes 5 numbers from user
- Finds greatest, second greatest, smallest
- Counts even numbers
- Handles: negatives, duplicates, all same numbers
